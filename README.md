# GNSS Processing GUI

Desktop-приложение на Python/PySide6 для импорта, визуализации и обработки GNSS/GPS временных рядов высот и связанных данных. Репозиторий содержит исходный код, скрипты сборки и тестовые файлы; готовые бинарные сборки публикуются через GitHub Releases. [web:1034][web:1022]

## Назначение

Программа предназначена для:
- импорта GNSS/GPS данных из файлов;
- первичной обработки и фильтрации временных рядов;
- визуализации результатов;
- экспорта обработанных данных и журналов обработки.

## Зависимости

Основные зависимости:
- Python 3.x
- PySide6
- pandas
- numpy
- matplotlib
- openpyxl

Установка зависимостей:

```bash
pip install -r requirements.txt
```

## Запуск из исходников

Из корня проекта:

```bash
python run_gui.py
```

или:

```bash
python packaging/run_gnss_gui.py
```

## Сборка релиза

Сборка Windows-релиза выполняется из корня проекта:

```bash
python packaging/build_release.py
```

Дополнительно можно использовать bat-скрипт:

```bash
packaging/build_windows_release.bat
```

## Готовые сборки

Готовые `EXE` и `ZIP` не хранятся в истории git-репозитория, потому что GitHub рекомендует распространять бинарные артефакты через Releases, а не коммитить их в исходники. Скачивать готовые сборки нужно во вкладке **Releases** репозитория GitHub. [web:1034][web:1022]

## Импорт и тестовые файлы

Файлы для импорта и тестирования находятся в каталоге:

```text
data/input/
```

Результаты обработки и промежуточные выходные файлы обычно попадают в:

```text
data/output/
```

Эти выходные данные и сборочные артефакты не предназначены для хранения в git-репозитории.

## Структура проекта

```text
app/          # основной код приложения
app/core/     # импорт, фильтры, обработка, экспорт
app/gui/      # GUI на PySide6
app/plotting/ # графики и визуализация
data/input/   # входные и тестовые файлы
data/output/  # результаты обработки
docs/         # документация
packaging/    # spec, скрипты сборки, launcher
releases/     # локальные архивы релизов
```
